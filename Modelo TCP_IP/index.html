Qué es Modelo TCP/IP

El modelo TCP/IP es un conjunto de protocolos de red que permite la comunicación entre dispositivos en Internet y otras redes. Está compuesto por cuatro capas interconectadas, cada una con funciones específicas para garantizar una comunicación eficiente y confiable.

Capas del Modelo TCP/IP

Capa de Acceso a la Red

(Network Access Layer): Se encarga de la transmisión física de datos dentro de una red local.

Capa de Internet

(Internet Layer): Proporciona el enrutamiento y direccionamiento de paquetes entre redes.

Capa de Transporte

(Transport Layer): Ofrece servicios de transporte confiable o no confiable entre aplicaciones en los hosts.

Capa de Aplicación

(Application Layer): Permite la comunicación entre aplicaciones y usuarios, definiendo protocolos específicos para cada servicio.
